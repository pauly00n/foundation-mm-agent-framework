## Blackbox Added Memories
- Project: blackbox-mm-prototype — Autonomous ML experiment framework for cardiac MRI classification (ACDC dataset, 5 classes: NOR/DCM/HCM/MINF/RV). Structure: src/prepare.py (fixed data pipeline), src/train.py (agent iterates on this), program.md (agent instructions), outputs/results.jsonl (experiment log), outputs/research_log.md (agent reasoning log). Stack: PyTorch, nibabel, scikit-learn, tqdm, uv for dependency management. GPU: NVIDIA H100 80GB. Volume shape: (1, 16, 128, 128). Baseline model: 4-block 3D CNN, 1.16M params. Training budget: 10 min wall-clock (BUDGET_SECONDS=600, never change). Run order: uv sync → uv run src/prepare.py → uv run src/train.py → blackbox program.md.
- Training budget changed to 1 minute (BUDGET_SECONDS=60) in src/train.py.
